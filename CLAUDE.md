# Just Beyond The Bend — Claude Code Project Guide

## Project Overview
Marketing / holding company site for Just Beyond The Bend LLC (Culver City / West LA area).
This is a clean, professional site that represents the LLC as a holding company.

## Domain
- **Production:** justbeyondthebend.com (managed via Cloudflare DNS → Vercel)

## Tech Stack
- **Framework:** Astro (static output)
- **Styling:** Tailwind CSS v4 (via `@tailwindcss/vite`)
- **Package manager:** npm

## Dev Commands
```bash
npm install          # Install dependencies
npm run dev          # Dev server at http://localhost:4321
npm run build        # Production build → dist/
npm run preview      # Preview production build locally
```

## Notes
- Separate from the Reading App project (goodreading.co)
- Vercel project will be created fresh for this domain
